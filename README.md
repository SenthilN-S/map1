# CrowdSafe Platform

Real-time crowd intelligence and accessible navigation system.

## Deployment Links
- **User Dashboard**: [https://crowdsafe-user.vercel.app](https://crowdsafe-user.vercel.app)
- **Admin Dashboard**: [https://admin-dashboard-five-alpha.vercel.app](https://admin-dashboard-five-alpha.vercel.app)

## Features
- Real-time heatmaps for crowd density.
- Safe routing to avoid overcrowded areas.
- SOS emergency alerts with user identity (Name & Phone).
- Administrative control panel for monitoring and configuration.
